# Spec
## Version: 0.1.0
## Last updated: 2026-04-05 - Initial project scaffold
## Project: GitKit

Detailed technical specifications for the GitKit Manager App.

### Core Features
- Committing & Staging (working tree diff, stage/unstage, commit)
- Commit Graph Visualization (GitKraken Style using D3 SVG, virtualized)
- Branch Management (list, create, checkout, delete, rename, setting upstream)
- Remote Operations (Fetch, Pull, Push with real-time streaming to UI)
- Stash Visualization (List, apply, drop, inline diff tracking)

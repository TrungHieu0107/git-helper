# 05 - Workflow: New Feature
## Version: 1.0.0
## Last updated: 2026-04-25 – Initial documentation setup
## Project: GitKit (git-helper)

1. **Understand Requirements**: Full understanding of user needs and modern design inspirations.
2. **Implementation Plan**: Create `implementation_plan.md` and obtain user approval.
3. **Core Development**: Start with CSS/Design tokens, then components, then assembly.
4. **Virtualization Audit**: Ensure new UI elements don't degrade scroll performance.
5. **Documentation**: Update `02-features.md` and `04-current-code.md`.
6. **Verification**: Run build checks and manual UI verification.

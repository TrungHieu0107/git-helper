# 06 - Workflow: Fixing a Bug
## Version: 1.0.0
## Last updated: 2026-04-25 – Initial documentation setup
## Project: GitKit (git-helper)

1. **Reproduce & Isolate**: Confirm the bug and its location.
2. **Minimal Fix**: Change only necessary lines with English + Vietnamese comments.
3. **Strict Testing**: Verify the fix and ensure no regressions in core components (Graph, Diff).
4. **Immediate Update**: Update `04-current-code.md` and `bug_registry.md`.
5. **Commit**: Use `fix: [area] - [description]` format.

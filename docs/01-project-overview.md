## Project: GitKit (git-helper)

GitKit is a modern, high-performance Git GUI built with Tauri v2, React, and Rust. It features a virtualization-based commit graph, powerful conflict resolution, and a high-density Dracula-themed UI. It is designed to handle massive repositories (1M+ commits) with sub-second responsiveness.

# GitManager App Summary
## Version: 1.0.0
## Last updated: 2026-04-05 - Manhattan-Routed Commit Graph
## Project: GitKit

GitKraken-style desktop Git client. Commit graph uses Manhattan routing (horizontal→vertical connections) refined with rounded corners via SVG arcs. Single continuous SVG with background vertical lane lines. Regular commits: large circle nodes (r=14) with avatar initials. Merge commits: small filled dots (r=5). Branch labels colored by lane. Resizable columns. Custom scrollbars.

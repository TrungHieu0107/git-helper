# Git Helper Client
The Git Helper application features a high-fidelity, GitKraken-style commit graph, robust HTTP authentication using Git Credential Manager, and a robust "vibe coding" right panel logic. The app features a global Monaco-powered Diff Viewer supporting Side-by-Side patch viewing, and a recently improved branch label system in the commit graph that groups multiple branches into a single primary badge with a checkout-enabled dropdown for additional refs. Selection is now globally synced, allowing the "View Changes" button in the right panel to seamlessly navigate back to the current WIP state.

# GitKit Status Summary
## Version: 2.5.0
## Last updated: 2026-04-11 – Added File Context Menu and History Modal
## Project: GitKit

GitKit now supports professional file-level operations via right-click and a comprehensive, searchable File History Modal with integrated Monaco-based diffing.

**Recent Highlights**:
- **File Context Menu**: Right-click any file in the Right Panel to open in editor, reveal in explorer, stage/unstage, or discard changes.
- **File History Modal**: A full-screen overlay to explore the complete history of any file, with commit list and searchable path input.
- **Safe Discard**: Intelligent discard logic that handles both tracked and untracked files safely.
- **Portability**: Refactored diff rendering to support both working tree and historical commit views.

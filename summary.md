# Git Helper Client
The Git Helper application features a high-fidelity, GitKraken-style commit graph, robust HTTP authentication using Git Credential Manager, and a robust "vibe coding" right panel logic. Recently, the app has been significantly upgraded to feature a global Monaco-powered Diff Viewer supporting Side-by-Side patch viewing, with advanced `encoding_rs` backend bindings for flawlessly viewing Vietnamese and Japanese local file data encodings.

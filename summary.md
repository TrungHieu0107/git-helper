# GitManager App Summary
## Version: 0.9.0
## Last updated: 2026-04-05 - Single SVG Graph + Resizable Columns
## Project: GitKit

GitKraken-style desktop Git client. Commit graph uses a single continuous SVG with bezier curves (M x1 y1 C x1 yMid x2 yMid x2 y2). Nodes show avatar initials inside colored circles. All table columns (Label, Hash, Author) are drag-resizable. Custom scrollbars applied globally. Sidebar derives branches from commitLog refs.

# Summary
## Version: 1.2.2
## Last updated: 2026-04-09 – Visual Polish
## Project: GitKit

The Git Helper application features a high-fidelity, GitKraken-style commit graph with Manhattan routing, Advanced Branching, and Inline Stash visualization. Stashes are integrated as square nodes with dedicated, isolated lanes and smooth connection lines. The UI is dense and professional, with resizable columns, grouped branch labels, and a polished sidebar. Recent additions include real-time name validation, smart auto-stashing, and right-click context menus. Other features include a dynamic Repository Selector, infinite scrolling, and a Monaco-powered Diff Viewer.

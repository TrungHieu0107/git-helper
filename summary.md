# GitKit Summary
## Version: 2.3.0
## Last updated: 2026-04-25 – Compact UI & Design System Standard
## Project: GitKit

GitKit has been refactored with a high-density "Compact UI" design system, standardizing on a 5-10px spacing scale (px-2.5, py-1.5, gap-1.5). The commit row height has been adjusted to 29px (compact) and 37px (normal) to provide better legibility while maintaining a streamlined IDE aesthetic.

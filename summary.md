# Git Helper Client
The Git Helper application features a high-fidelity, GitKraken-style commit graph with Manhattan routing and an Advanced Branching System. Users can create branches in three modes: Local Only, Create + Push (with upstream tracking), and From Remote. Features include real-time name validation, automatic stashing for dirty trees, and a searchable source selector. Right-clicking any commit opens a context menu for branch creation, checkout, and commit data copying. The UI supports global state sync, Monaco-powered diffs, and dynamic Repo switching.
